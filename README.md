# MDD665 ESPHome

Integracja sterownika MDD665 z ESPHome.

## Szybki start z ESPHome Dashboard

W ESPHome kliknij `New Device` → `Continue` → `Import` i użyj tego URL:

```yaml
dashboard_import:
  package_import_url: https://raw.githubusercontent.com/SQ9MDD/mdd665-esphome/main/mdd665_default.yaml
  import_full_config: false
